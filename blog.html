What are the different ways to select an element in the DOM?

Different ways, getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), or querySelectorAll().

What is the difference between innerHTML, innerText, and textContent?

innerHTML lets you get or set HTML content, innerText gives you only visible text, and textContent gets all text, even the hidden ones.

What is event delegation in the DOM?

It’s a technique where a parent element listens for events on its child elements instead of adding event listeners to each child.

What is event bubbling in the DOM?

It’s when an event starts at the target element and moves up to its parent elements.

How do you create, add, and remove elements using JavaScript?

Use createElement() to make an element, appendChild() or append() to add it, and removeChild() or remove() to delete it.
