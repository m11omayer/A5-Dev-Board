What are the different ways to select an element in the DOM?

Different ways to select an element in the DOM: getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll().

What is the difference between innerHTML, innerText, and textContent ?

Difference between innerHTML, innerText, and textContent: innerHTML gets/sets HTML content, innerText gets/sets visible text, and textContent gets/sets all text including hidden elements.

What is event delegation in the DOM?

Event delegation in the DOM: A technique where a parent element handles events for its child elements using event bubbling.

What is event bubbling in the DOM?

Event bubbling in the DOM: A process where an event starts from the target element and propagates up to its ancestors.

How do you create, add, and remove elements using JavaScript?

Creating, adding, and removing elements using JavaScript: Use document.createElement(), appendChild(), removeChild(), append(), and remove().
